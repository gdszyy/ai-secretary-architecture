# 赛季通行证活动方案

> **文档类型**: PRD  
> **来源**: https://kjpp4yydjn38.jp.larksuite.com/wiki/BE2KwvGDKiu1AQkZQwcj87WHpId  
> **归属模块**: `mod_activity`  

## 业务目标

通过完成任务获取经验值提升等级，解锁免费及高级奖励，提升用户活跃度和付费转化。

## 功能详情

### 1. 等级状态 (FP-ACTIVITY-001)

**功能描述**: 动态展示用户当前等级及经验值进度，提示本周经验获取情况。

**业务规则**:
- 显示当前等级和当前经验值/下一级经验值
- 展示本周经验获取进度及上限
- 未登录状态使用模拟数据，所有交互弹出登录注册弹窗

### 2. 奖励板块 Rewards Track (FP-ACTIVITY-002)

**功能描述**: 用户可一键领取或单独领取已解锁奖励，区分免费和高级奖励样式。

**业务规则**:
- 点击“CLAIM ALL”领取所有已解锁未领取奖励
- 奖励支持横向滑动展示，移动端为竖向
- 免费和高级奖励UI视觉样式差异化
- 图标优先读取对应游戏封面，失败则使用默认设计图标
- 领取状态分为可领取（红点提示）、已领取（绿色勾选）、未解锁（锁头图标）
- 高级通行证付费按钮根据优惠状态显示价格，支持余额不足提示及充值引导
- 已开通用户隐藏升级卡片

### 3. 任务卡片 Missions (FP-ACTIVITY-003)

**功能描述**: 展示每日任务和每周任务，实时更新进度，完成后自动发放经验值。

**业务规则**:
- 任务分为每日任务和每周任务
- 进度条实时更新
- 任务完成后自动发放经验值，无需手动领取

### 4. 准入逻辑 (FP-ACTIVITY-004)

**功能描述**: 限制活动参与用户，确保只有符合条件的用户可参与。

**业务规则**:
- 仅限历史投注笔数≥1的用户参与
- 冻结用户等拦截标签禁止参与及停止活动相关逻辑

### 5. 等级与经验值计算逻辑 (FP-ACTIVITY-005)

**功能描述**: 通过累计经验值提升等级，等级采用经验值阈值制，支持周经验值上限及解除。

**业务规则**:
- 用户等级由累计总经验值决定，达到阈值升级
- 周经验值获取有上限，超过上限不再增加经验
- 超过解除日期后取消周经验值上限限制
- 经验值活动期间永久累计，不重置

### 6. 奖励发放与领取逻辑 (FP-ACTIVITY-006)

**功能描述**: 奖励分免费和高级双轨，用户达成等级条件后解锁，点击领取后系统发放。

**业务规则**:
- 免费奖励等级达成自动解锁
- 高级奖励需等级达成且购买Premium通行证
- 领取状态分未解锁、可领取、已领取
- 支持一键领取所有已解锁未领取奖励
- 根据奖励类型调用对应礼券接口，数值严格对应后台配置

### 7. Premium增值服务支付逻辑 (FP-ACTIVITY-007)

**功能描述**: 用户通过账户余额购买一次性Premium权限，支持优惠价及余额不足提示。

**业务规则**:
- 支持限时优惠价，优惠时间内自动切换售价
- 购买需登录且满足准入条件
- 余额不足时提示并引导充值
- 购买成功后立即生效，权限具有追溯性
- Premium身份活动周期内唯一，不可重复购买

### 8. 任务系统逻辑 (FP-ACTIVITY-008)

**功能描述**: 任务为经验值唯一产出渠道，自动判定完成条件并发放经验值。

**业务规则**:
- 每日任务按日周期判定，实时累加行为数据
- 每周任务按周周期判定，维护连续天数计时器
- 仅计算满足赔率≥1.60且已结算的体育注单
- 取消、退回或走盘注单不计入任务

### 9. 回滚逻辑 (FP-ACTIVITY-009)

**功能描述**: 注单状态变更导致经验值和任务贡献回滚，处理不同场景的经验值和奖励状态。

**业务规则**:
- 非违规取消注单扣除对应经验值
- 用户降级但未领奖时锁定奖励，后续升级正常解锁
- 用户降级且已领奖时仅回滚经验值，不追回奖励
- 违规取消注单直接取消活动资格
