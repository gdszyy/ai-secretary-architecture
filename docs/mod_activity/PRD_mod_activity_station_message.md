# 站内信系统需求说明

> **文档类型**: PRD  
> **来源**: https://kjpp4yydjn38.jp.larksuite.com/docx/CLD6dtAmjoj915x2vBkjYLhHphh  
> **归属模块**: `mod_activity`  

## 业务目标

通过站内信系统向用户推送多样化消息，提升信息接收效率与活动参与度。

## 功能详情

### 1. 新增 (FP-ACTIVITY-001)

**功能描述**: 建立站内信内容，包括一般站内信、含奖励站内信及推广站内信。

**业务规则**:
- 一般站内信配置发送时间，时间到则发送
- 含奖励站内信可配置回收时间
- 推广站内信特殊处理

### 2. 查询 (FP-ACTIVITY-002)

**功能描述**: 根据条件查询已建立的站内信内容。

**业务规则**:
- 支持标题查找
- 支持内容查找
- 支持发送时间区间查找
- 支持奖励金额查找
- 支持是否为广告信筛选

### 3. 重送 (FP-ACTIVITY-003)

**功能描述**: 将原发送的站内信重新发送。

**业务规则**:
- 重新发送单条信息
- 依配置内容发送

### 4. 站内信发送记录 (FP-ACTIVITY-004)

**功能描述**: 依条件查询站内信发送记录。

**业务规则**:
- 支持标题查找
- 支持发送时间区间查找
- 支持奖励金额查找
- 支持过期未领取奖励查找
