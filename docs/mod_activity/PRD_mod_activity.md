# 用户推广奖励机制需求文档

> **文档类型**: PRD  
> **来源**: https://kjpp4yydjn38.jp.larksuite.com/docx/LiYRdVnrEoPNRJxhKSTjrzoVpUg  
> **归属模块**: `mod_activity`  

## 业务目标

通过建立用户推广奖励机制，激励用户分享邀请链接，带动新用户注册及参与，实现低成本高效用户增长。

## 功能详情

### 1. 活动配置 (FP-ACTIVITY-001)

**功能描述**: 配置用户推广活动的基础信息、参数、奖励策略及用户筛选，支持活动发布与管理。

**业务规则**:
- 活动名称、活动时间、预算来源为必填项，未填时展示toast提示
- 支持配置单个优质用户条件及里程碑激励条件，条件可增删
- 奖励策略中，奖励类型从模板选择，转账限制当前固定100%，未来可能调整
- 发布时支持邀请码生成方式选择（随机或依帐号），活动开始后不可修改时间范围

### 2. 代理奖励派发规则 (FP-ACTIVITY-002)

**功能描述**: 实时派发代理奖励，达成目标后代理详情页领取按钮点亮。

**业务规则**:
- 奖励分为单个优质用户奖励和里程碑奖励
- 奖励需完成一倍流水后才能划转至主钱包
- 每月1日统计上月奖励，每月5日开启领取

### 3. 前台展示与分享功能 (FP-ACTIVITY-003)

**功能描述**: 展示用户邀请信息、奖励进度及分享邀请链接和图档，支持多平台分享。

**业务规则**:
- 个人信息模块展示邀请统计及邀请码，支持分享按钮调用多平台分享
- 阶梯奖励模块展示单个优质用户奖励及领取状态，完成任务发站内信通知
- 里程碑激励模块统计被邀请人完成情况，未领取奖励显示总额及领取提示
- 邀请人列表按充值+投注总额排序展示
- FAQ模块展示代理相关常见问题，未配置时显示静态页面

### 4. 全民代理分享组件 (FP-ACTIVITY-004)

**功能描述**: 提供多平台分享按钮及复制链接功能，支持原生分享和各平台分享链接生成。

**业务规则**:
- 分享内容包含标题、文本及邀请链接
- 支持Facebook、Telegram、WhatsApp、Kwai等平台分享链接生成
- 点击事件统一处理，原生分享优先，失败时提示用户
- 复制链接功能支持Clipboard API及兼容fallback方案

# 网页底部导航及移动端适配修改点

> **文档类型**: PRD  
> **来源**: https://kjpp4yydjn38.jp.larksuite.com/wiki/OctLwBiTciF0YQk0JIHjTW5ap0b  
> **归属模块**: `mod_activity`  

## 业务目标

优化网页底部导航及移动端页面布局和跳转体验，提升用户访问便捷性和界面适配性

## 功能详情

### 1. 网页底部导航跳转 (FP-ACTIVITY-001)

**功能描述**: 实现网页底部各模块页面的跳转功能

**业务规则**:
- Sports-Live Sport 跳转至Live Sport页面
- Sports-Sport Home 跳转至体育首页
- Sports-Sports Rules 跳转，默认展开Reglas Generales
- Casino-Casino Home 跳转至Casino首页
- Casino-Slots 跳转至Casino-slot页
- Casino-Promotions 跳转至promotion页
- Legal相关条款页面跳转（Terms of Service、Privacy Policy、AML/KYC Policy、Responsible Gaming）
- Support相关页面跳转（FAQ、Contact Us、Deposit & Withdrawal）
- 负责任博彩logo及链接放置于网站地图下方，18+跳转至Responsible Gaming界面

### 2. 移动端适配修改 (FP-ACTIVITY-002)

**功能描述**: 调整移动端页面间距、文本显示及导航结构，优化移动端用户体验

**业务规则**:
- 移动端间距调整为8，PC端为16
- 移动端隐藏超长文本
- 比赛卡片添加边框和分割线，隐藏快捷盘口和展开按钮
- 底部菜单新增“Live”和“Casino”，移除原promotion
- 顶部导航移除“比赛部分”面包屑，确保导航宽度适配
- 详情页对阵信息改为移动端版本
- 单specifier盘口超过三个选项改为一行一个
- 侧边栏导航二三级选项高度至少40，悬浮样式优化
- outright一行一个显示
- 个人中心菜单点击跳转，顶部添加返回按钮

### 3. 顶部导航宽度适配 (FP-ACTIVITY-003)

**功能描述**: 当宽度超过500时，顶部导航改为下方ICON版本，悬浮显示按钮文字

**业务规则**:
- 宽度超过500触发ICON版本导航
- 悬浮时显示按钮文字，具体交互见figma

### 4. 活动页跳转 (FP-ACTIVITY-004)

**功能描述**: 实现首页banner及充值banner跳转至活动页

**业务规则**:
- 首页banner跳转
- 充值banner跳转
