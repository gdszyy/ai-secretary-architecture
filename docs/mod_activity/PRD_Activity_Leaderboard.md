# 活动方案

> **文档类型**: PRD  
> **来源**: https://kjpp4yydjn38.jp.larksuite.com/wiki/Rq5IwVvZriXekWkp8VLj4DAmpCc  
> **归属模块**: `mod_activity`  

## 业务目标

通过配置多维度活动排名和限制规则，实现精准的活动管理和用户参与控制

## 功能详情

### 1. 活动排名统计 (FP-ACTIVITY-001)

**功能描述**: 支持实时总榜、日榜、周榜和月榜的排名统计与重置

**业务规则**:
- 实时总榜：整个活动周期累计排名，不重置
- 日榜：按自然日统计（00:00–23:59），每天重置
- 周榜：按自然周统计（周一00:00–周日23:59），每周重置
- 月榜：按自然月统计，每月重置

### 2. 活动模版配置 (FP-ACTIVITY-002)

**功能描述**: 提供活动基本信息、参数、奖励策略、用户客群及上线配置的模板设置

**业务规则**:
- 基本信息包含活动名称、时间、预算来源和备注
- 用户客群通过准入标签、拦截标签、准入UID和拦截UID进行控制
- 上线配置支持限制同一用户、设备ID、IP参与次数及未KYC认证账号限制

### 3. 礼券模版配置 (FP-ACTIVITY-003)

**功能描述**: 定义礼券的名称、类型及备注信息

**业务规则**:
- 礼券需配置名称、类型和备注
