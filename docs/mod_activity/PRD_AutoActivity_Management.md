# 自动化活动管理系统需求文档

> **文档类型**: PRD  
> **来源**: https://kjpp4yydjn38.jp.larksuite.com/docx/YppKd74NqohDdpxUe0Fjwd3UpDc  
> **归属模块**: `mod_activity`  

## 业务目标

构建一个高度可配置的活动管理平台，提升活动上线效率和运营管理能力

## 功能详情

### 1. 活动列表 (FP-ACTIVITY-001)

**功能描述**: 集中管理营销活动生命周期，支持多状态分类及预算和指标监控

**业务规则**:
- 支持通过Tab分类（进行中、暂停、已结束、草稿）快速查看活动进度
- 实时展示活动奖金的支出进度，帮助运营控制预算风险
- 活动状态与前端展示状态分开控制
- 上线时活动开始时间不能早于当前时间

### 2. 创建/编辑活动 (FP-ACTIVITY-002)

**功能描述**: 通过模板化步骤化流程完成活动创建及配置

**业务规则**:
- 活动模板一旦选定不可更改，决定后续参数和奖励策略
- 基础信息必填项包括活动名称和活动时间，时间不可早于当前时间
- 奖励策略支持礼券模板选择及金额、数量、最大返彩金额配置
- 用户筛选支持准入标签、拦截标签、自定义UID及互斥逻辑
- 风控限制包括用户参与次数、IP限制、设备ID限制及KYC认证

### 3. 查询面板 (FP-ACTIVITY-003)

**功能描述**: 支持多条件组合筛选活动数据

**业务规则**:
- 支持活动名称模糊匹配
- 支持活动模板单选筛选
- 支持活动时间范围筛选
- 查询条件组合为AND逻辑，需手动触发查询
