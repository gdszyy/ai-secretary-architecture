# 责任博彩机制需求说明

> **文档类型**: PRD  
> **来源**: https://kjpp4yydjn38.jp.larksuite.com/docx/EpzWdiGpVo6m8mxaM92j4L3gp7b  
> **归属模块**: `mod_activity`  

## 业务目标

确保平台符合巴西当地法规，落实责任博彩机制，保障用户身心健康与财务安全。

## 功能详情

### 1. 充值限额 (FP-ACTIVITY-001)

**功能描述**: 配置用户的充值额度上限，防止超额充值。

**业务规则**:
- 每一次修改，皆为隔周后生效
- 首次配置立即生效
- 充值总额度超过限额时，提示超额信息

### 2. 总损失限额 (FP-ACTIVITY-002)

**功能描述**: 配置用户的总损失上限，限制用户最大亏损额度。

**业务规则**:
- 每一次修改，皆为隔周后生效
- 首次配置立即生效
- 当周损失总额度超过限额时，提示超额信息

### 3. 游戏时间限制 (FP-ACTIVITY-003)

**功能描述**: 配置用户每日的游戏休息时间，保障合理游戏时长。

**业务规则**:
- 每日最少6小时的休息
- 每一次修改，皆为隔周后生效
- 登录时间若处于限制时间内，禁止登录并提示用户

### 4. 自我排除（Self-Exclusion） (FP-ACTIVITY-004)

**功能描述**: 允许用户设置自我封禁时间，限制其在封禁期间登录。

**业务规则**:
- 用户在自我封禁期间无法登录
- 封禁结束后方可登录

### 5. 后台解除限制 (FP-ACTIVITY-005)

**功能描述**: 后台可针对单一或全部用户解除责任博彩限制。

**业务规则**:
- 解除操作立即生效
- 支持单一解除和全部解除
- 前端需增加全部解除按钮

### 6. 后台展示－用户明细 (FP-ACTIVITY-006)

**功能描述**: 后台展示用户责任博彩配置详情，支持展开收起操作。

**业务规则**:
- 展示充值限额、总输限额、自我封禁时间
- 配置项展示在Gambline标签下
- 支持缩放展开与收回
