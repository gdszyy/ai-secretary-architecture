# 活动方案

> **文档类型**: PRD  
> **来源**: https://kjpp4yydjn38.jp.larksuite.com/wiki/OCf0wdnWPiXMxRk2li7jPvPnpjf  
> **归属模块**: `mod_activity`  

## 业务目标

通过VIP等级体系及奖励机制提升用户活跃度和忠诚度

## 功能详情

### 1. VIP状态区 (FP-ACTIVITY-001)

**功能描述**: 展示用户VIP等级信息及权益，支持等级进度和权益对比

**业务规则**:
- 未登录状态下不展示
- 显示当前用户VIP名称、层级及图标
- XP进度条根据有效流水转化比例自动更新
- 展示当前等级与下一等级的返水比例、晋级礼金、周红包数值
- 点击VIEW ALL LEVELS按钮弹窗展示前20个VIP等级，20级后显示coming soon
- 点击跳转SVIP介绍页

### 2. Rewards Center (FP-ACTIVITY-002)

**功能描述**: 展示用户可领取的各类奖励卡片及领取状态

**业务规则**:
- 未登录状态下不展示
- 奖励类型包括Sport/Casino Cashback、Level Up Bonus、Weekly Bonus、Birthday Bonus
- 领取按钮状态根据是否可领取及时间限制高亮或置灰
- 领取提示显示奖励失效时间、可领取时间或等级限制
- 时间格式分为≥1天、≥1小时及<1小时三种显示方式

### 3. TIER BENEFITS (FP-ACTIVITY-003)

**功能描述**: 展示VIP各等级权益的静态表格及详细弹窗

**业务规则**:
- 表格行显示权益名称，列显示VIP分组名称
- 点击Detail按钮弹窗展示详细数据，数据由后台读取

### 4. FAQ (FP-ACTIVITY-004)

**功能描述**: 提供手风琴式展开收起的常见问题解答

**业务规则**:
- 交互形式为手风琴式展开/收起

### 5. 准入逻辑 (FP-ACTIVITY-005)

**功能描述**: 控制用户是否准入活动及相关逻辑处理

**业务规则**:
- 默认所有用户准入
- 冻结用户等被后台配置拦截，禁入活动及停止所有相关逻辑处理

### 6. XP经验值计算逻辑 (FP-ACTIVITY-006)

**功能描述**: 根据用户有效流水计算VIP经验值

**业务规则**:
- XP累计每月不重置
- Free Bet、Free Spin、Bonus不计入VIP累计流水，仅cash计入
- 无效注单或重新结算时扣除相应XP
- 体育有效流水：赔率≥1.6x投注金额计1 XP
- 游戏有效流水按不同游戏类型乘以比例计XP
- 后台经验值配置变更不影响用户当前经验值

### 7. 晋级保级降级逻辑 (FP-ACTIVITY-007)

**功能描述**: 定时判断用户VIP等级的升级、保级和降级

**业务规则**:
- 每月1号00:00定时判断
- 晋级判定优先于保级判定
- 晋级判定：XP达到下一级阈值立即升级
- 保级判定：统计上月总XP，低于保级经验则降级
- 降级时XP重置到目标等级起始值

### 8. 权益发放逻辑 (FP-ACTIVITY-008)

**功能描述**: 根据VIP等级定期发放各类奖励

**业务规则**:
- 体育及游戏返水每周一结算，按当时返水比例计算
- 晋级礼金在升级事件触发时累计奖励池，一周内多级合并计算
- 每周红包按VIP等级每周发放
- 生日礼金限生日当天领取，暂不开发
- 提款额度、提款次数、专属客服暂不开发
- Cashback、Level Up Bonus、Weekly Bonus每周一统计上周奖励，每周四13:00至周日23:59:59开放领取，逾期自动失效
- 生日礼金当天0点开放领取，逾期自动失效
