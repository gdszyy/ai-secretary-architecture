# 活动方案

> **文档类型**: PRD  
> **来源**: https://kjpp4yydjn38.jp.larksuite.com/wiki/DibDwTKWBiWpbWkmcfajhuXrpyb  
> **归属模块**: `mod_activity`  

## 业务目标

通过多样化任务和抽奖机制提升用户活跃度和参与度，促进用户留存和转化。

## 功能详情

### 1. 头部信息 (FP-ACTIVITY-001)

**功能描述**: 显示活动标题及活动有效期。

**业务规则**:
- 显示活动标题“Daily Draw · Continuous Rewards”
- 显示活动有效期

### 2. 抽奖按钮 (FP-ACTIVITY-002)

**功能描述**: 用户通过抽奖券进行抽奖，按钮展示券数及状态反馈。

**业务规则**:
- 按钮右上角气泡展示当前拥有的抽奖券数量
- 提示抽奖券有效期为3天内有效
- 点击按钮时检查抽奖券余额，余额大于0进入抽奖流程，余额为0按钮抖动并弹出提示
- 抽奖成功后按钮变为禁用状态，防止重复扣券或请求重叠

### 3. 任务面板 (FP-ACTIVITY-003)

**功能描述**: 展示每日投注任务、补签任务和邀请任务，用户完成任务可获得抽奖券奖励。

**业务规则**:
- 每日投注任务每日限领1张抽奖券，0点自动重置
- 补签任务允许用户通过累计投注金额补领漏签的抽奖券，分阶段发放
- 邀请任务实时显示邀请人数，点击可打开邀请分享弹窗

### 4. 奖池展示 (FP-ACTIVITY-004)

**功能描述**: 展示奖品图标及抽奖状态，配合抽奖动画提升用户体验。

**业务规则**:
- 不同奖品使用对应图标，缺失时使用默认设计图标
- 已抽取奖品覆盖绿色勾选框，未抽取正常显示
- 抽奖动画分为起势、循环加速和减速定格三个阶段，时长控制在2.5-3.5秒

### 5. 邀请分享弹窗 Invite&Earn (FP-ACTIVITY-005)

**功能描述**: 提供专属邀请链接和邀请码，支持一键复制及社交媒体快捷分享。

**业务规则**:
- 复用代理分享活动里的邀请模块
- 自动生成带二维码的邀请海报，用户可保存分享

### 6. 中奖概率弹窗 Prize Probabilities (FP-ACTIVITY-006)

**功能描述**: 以表格形式展示所有奖品及其中奖权重，告知中奖概率计算公式。

**业务规则**:
- 展示奖品名称及对应权重
- 概率计算公式为该奖品权重除以总剩余权重乘以100%

### 7. 站内信通知 (FP-ACTIVITY-007)

**功能描述**: 系统自动发放抽奖券时，通过站内信通知用户并支持跳转活动页。

**业务规则**:
- 自动发放抽奖券时触发站内信通知
- 点击站内信banner跳转活动页面

### 8. 规则信息 Rules & Info (FP-ACTIVITY-008)

**功能描述**: 展示活动相关的静态规则信息页面。

**业务规则**:
- 静态页面展示规则信息
