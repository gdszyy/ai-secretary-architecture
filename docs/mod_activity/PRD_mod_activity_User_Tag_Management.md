# 用户标签管理功能需求说明

> **文档类型**: PRD  
> **来源**: https://kjpp4yydjn38.jp.larksuite.com/docx/GsxNdQ8Noo0bssxN5vqjP8nipad  
> **归属模块**: `mod_activity`  

## 业务目标

通过标签化管理平台用户，实现动态控制用户权限和资源领取。

## 功能详情

### 1. 用户标签配置 (FP-ACTIVITY-001)

**功能描述**: 配置和管理用户标签，实现用户的标签化管理。

### 2. 用户标签配置－新增 / 修改 (FP-ACTIVITY-002)

**功能描述**: 新增或修改用户标签信息。
