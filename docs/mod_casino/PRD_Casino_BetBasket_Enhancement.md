# 产品设计文档 (PRD)

> **文档类型**: PRD  
> **来源**: https://kjpp4yydjn38.jp.larksuite.com/wiki/YC0Dw4Sfjim2WQkbhXkjRUxfpb0  
> **归属模块**: `mod_casino`  

## 业务目标

提升投注篮的视觉吸引力和操作效率，优化PC端和移动端的交互体验。

## 功能详情

### 1. 投注工具栏与基础样式调整 (FP-CASINO-001)

**功能描述**: 调整设置按钮样式，新增“Bet on all”按钮组，实现快速统一设置所有注单金额。

**业务规则**:
- 设置按钮样式微调，具体以Figma设计稿为准。
- “Bet on all”按钮组默认包含50、100、500三个数值，可配置。
- 点击“Bet on all”按钮后，所有注单金额统一更改对应数值。
- 串关模式下，操作直接修改串关投注金额。

### 2. 串关（Parlay）模式视觉优化 (FP-CASINO-002)

**功能描述**: 优化串关模式视觉效果，增加小问号解释图标，调整投注金额底色。

**业务规则**:
- Parlay右侧添加小问号图标，点击展示串关逻辑说明，常驻且single模式下也显示。
- 串关投注金额底色改为偏灰色以增强视觉区分。

### 3. 投注卡片交互与样式修改（区分PC端与移动端） (FP-CASINO-003)

**功能描述**: 调整投注卡片输入框和按钮样式，新增累加按钮及其交互，区分PC端和移动端布局与交互细节。

**业务规则**:
- 输入框和按钮样式整体调整，具体以Figma设计稿为准。
- “Bet on all”按钮修改金额时，输入框出现红色脉冲动画并变红。
- 单注卡片设有三个本金投注金额累加按钮，数值可配置，点击累加对应金额。
- PC端布局为输入框左侧，累加按钮右侧同一行；移动端为上下结构，输入框上方，累加按钮下方。
- PC端默认浅色边框，悬浮卡片组件边框变深灰，悬浮按钮边框变红。
- 鼠标悬浮累加按钮时，输入框左侧预览将添加数值。
- 点击累加按钮出现红色加额提示动画、数字滚动跳转动画及输入框红色脉冲。
- 移动端键盘唤起时隐藏工具栏，注单量少时排序自下而上底部对齐。
- 移动端投注篮以抽屉形式自下而上展现，顶部预留蒙层区域，滚动至顶部显示返回按钮。

### 4. 设置 (FP-CASINO-004)

**功能描述**: 提供Quick buy和Bet-on-all金额配置，支持悬浮提示气泡和引导用户设置。

**业务规则**:
- Quick buy amount配置三个加注按钮数值，悬浮时弹出提示气泡。
- Bet-on-all amount配置投注工具栏三个按钮数值，悬浮时弹出提示气泡。
- 未修改设置时，鼠标悬浮对应组件弹出引导气泡，带跳转链接高亮对应设置模块。
- 引导气泡文案支持多语言（英文、葡萄牙语、西班牙语）。
