# 优化清单

> **文档类型**: PRD  
> **来源**: https://kjpp4yydjn38.jp.larksuite.com/docx/AvN2dGjpNoGzpLxsl8VjAq4UpZc  
> **归属模块**: `mod_casino`  

## 业务目标

提升游戏大厅及游戏相关功能的用户体验和管理效率

## 功能详情

### 1. GameTag List (FP-CASINO-001)

**功能描述**: 增加Lobby下拉选择，支持根据Lobby筛选游戏标签

**业务规则**:
- 查询条件加上Lobby下拉选择
- 选择相关Lobby后，只展示该Lobby下的Tags

### 2. Game List (FP-CASINO-002)

**功能描述**: 新增Sort和RTP字段，支持列表、新增、编辑及导入操作

**业务规则**:
- 列表、新增、编辑、import均支持Sort和RTP字段
- Sort未填时默认值为1
- RTP未填时默认值为-

### 3. 游戏大厅 (FP-CASINO-003)

**功能描述**: 大厅数量过多时支持滑动展示

**业务规则**:
- 当大厅超过页面宽度时，支持滑动查看其他大厅内容

### 4. 游戏Tags (FP-CASINO-004)

**功能描述**: 增加游戏快速查找功能，支持多条件筛选和滑动展示

**业务规则**:
- 支持单一或合并查找
- 游戏商查找使用下拉选单，支持All和单一游戏商筛选
- 输入框支持模糊查找游戏名称
- Tags超过页面宽度时支持滑动

### 5. 游戏详情 (FP-CASINO-005)

**功能描述**: 新增FreeSpin功能及快速切换游戏，优化内嵌游戏展示和全频切换

**业务规则**:
- FreeSpin按钮展示及注册免费游戏
- 游戏中可快速切换其他游戏
- 内嵌游戏直接在示意图框中展示，不跳开展示
- 全频展示支持放大缩小切换
