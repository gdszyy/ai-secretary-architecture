# 游戏大厅入口与游戏管理系统设计文档

> **文档类型**: PRD  
> **来源**: https://kjpp4yydjn38.jp.larksuite.com/docx/ZeoMdbcWOoFLXfx2iVTj8GVEp7f  
> **归属模块**: `mod_casino`  

## 业务目标

实现游戏大厅入口、游戏类型、游戏标签及游戏管理的后台配置与前台展示，提升游戏展示与管理效率。

## 功能详情

### 1. 游戏大厅入口管理 (FP-CASINO-001)

**功能描述**: 后台配置游戏大厅入口，前台展示对应入口及相关游戏数据。

**业务规则**:
- 新增大厅后默认不启用，启用后前台展示入口
- Lobby Code 唯一且不可修改
- 启用状态控制前台是否展示入口
- 删除仅允许无绑定游戏类型或游戏的大厅
- 修改排序影响前台展示顺序

### 2. 游戏类型管理 (FP-CASINO-002)

**功能描述**: 管理游戏类型，绑定大厅入口，控制游戏类型启用状态及展示。

**业务规则**:
- 新增游戏类型默认不启用
- 游戏类型不启用时，绑定该类型的游戏前台不展示
- 删除仅允许无绑定游戏的类型
- 游戏类型绑定 Sport 大厅时不展示游戏
- 新增大厅入口时，游戏类型默认采用 Casino 排版

### 3. 游戏标签管理 (FP-CASINO-003)

**功能描述**: 管理游戏标签，支持自动新增对应三方游戏的 GameType 标签。

**业务规则**:
- 新增标签默认不启用，启用后前台展示对应坑位
- 删除仅允许无绑定游戏的标签
- 三方游戏新增 GameType 时自动新增对应标签
- 标签展示排版后续根据需求调整

### 4. 游戏管理 (FP-CASINO-004)

**功能描述**: 管理平台游戏信息及展示标签，支持游戏状态及维护时间配置。

**业务规则**:
- 游戏状态默认启动，支持维护时间配置
- 游戏标签预设带入 GameType 标签
- 游戏代码唯一且不可修改
- 支持游戏封面、地址及排序配置
- 绑定游戏类型控制游戏分类展示
