# 钱包范围与流程说明

> **文档类型**: TECH_SPEC  
> **来源**: https://kjpp4yydjn38.jp.larksuite.com/docx/MsjidsjeLoS6GaxyOqwjcORfpRd  
> **归属模块**: `mod_settlement`  

## 业务目标

规范用户钱包及相关资金流动行为，支持多币别及活动流水限制管理

## 功能详情

### 1. 资金流动行为管理 (FP-SETTLEMENT-001)

**功能描述**: 定义用户钱包及平台钱包在充值、提款、投注及返奖等操作中的资金变动规则

**业务规则**:
- 充值时用户主钱包余额增加，平台钱包无异动
- 提款时用户主钱包余额减少，平台钱包无异动
- 投注时用户主钱包余额减少，平台主钱包余额增加
- 返奖或取消时用户主钱包余额增加，平台主钱包余额减少
- SportBonus和FreeSports分别在用户和平台钱包间进行增减操作
- FreeSport反奖时盈利部分增加用户主钱包，平台主钱包相应减少

### 2. 钱包数据表设计 (FP-SETTLEMENT-002)

**功能描述**: 设计钱包及相关活动流水限制的数据库表结构，支持多币别及流水限制规则

**业务规则**:
- 钱包表包含钱包ID、帐号ID、币别、币别类型、余额、冻结金额等字段
- 币别类型支持Main、SportBonus、CasinoBonus、FreeSpin、FreeSport五种类型
- 用户参与活动明细表关联帐号和活动信息，记录参与条件
- 用户参与活动流水限制明细表记录流水限额规则及状态，支持循环模式和时间有效期
