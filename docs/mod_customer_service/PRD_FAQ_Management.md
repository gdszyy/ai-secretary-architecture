# 问与答（FAQ）管理功能需求说明

> **文档类型**: PRD  
> **来源**: https://kjpp4yydjn38.jp.larksuite.com/docx/OAx8dQZPKosnwkxDsZXjoax0p0f  
> **归属模块**: `mod_customer_service`  

## 业务目标

提供后台统一管理FAQ内容并在前台多情境展示，提升客服效率和用户体验。

## 功能详情

### 1. 查询FAQ (FP-CUSTOMER_SERVICE-001)

**功能描述**: 支持根据标题、内容和状态条件模糊或精准查找FAQ，并支持多字段排序。

**业务规则**:
- 标题、内容支持模糊查找
- 状态支持精准查找
- 排序支持点击字段切换正序、倒序和默认排序

### 2. 新增 / 修改FAQ (FP-CUSTOMER_SERVICE-002)

**功能描述**: 支持后台新增和编辑FAQ条目，包含多语言和多展示位置设置。

**业务规则**:
- 新增时可动态增加填写条目
- 支持删除单条FAQ数据
- 支持填写标题、关键字、问题、解答、展示位置、语言、排序等字段
- 语言支持简中、英文、巴西语、墨西哥语
- 一般FAQ和活动FAQ的标题及展示位置选项不同

### 3. 删除 / 停用 FAQ (FP-CUSTOMER_SERVICE-003)

**功能描述**: 支持启用/禁用FAQ及物理删除操作，删除需二次确认。

**业务规则**:
- 列表中通过开关切换启用/禁用状态
- 删除操作弹出确认框，确认后物理删除

### 4. Import (FP-CUSTOMER_SERVICE-004)

**功能描述**: 支持通过Excel文件批量导入FAQ，支持新增和修改。

**业务规则**:
- 带Id为修改，未带Id为新增
- 修改时仅调整有填写的字段，空字段不变

### 5. Export (FP-CUSTOMER_SERVICE-005)

**功能描述**: 支持导出符合查询条件的FAQ数据，便于批量修改后再导入。

**业务规则**:
- 导出数据包含Id
- 导出后可调整数据再导入实现批量修改

### 6. 前台展示 (FP-CUSTOMER_SERVICE-006)

**功能描述**: 根据后台配置的展示位置和分类，在不同页面和区块展示FAQ内容。

**业务规则**:
- 展示位置包括全站、VIP、客服、代理、充值、提款及活动详情区
- FAQ按大标题分组分类展示
- 支持对问题和解答内容的模糊查找
